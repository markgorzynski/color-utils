    Ciecam16ViewingConditions

    Properties:
    Name Type Attributes Default Description
    adaptingLuminance number

    Adapting field luminance in cd/m^2 (L_A).

    Luminance factor of the background (Y_b), as a percentage of reference white Y (e.g., 20 for 20% gray, 100 for white).
